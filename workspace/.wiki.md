# Project Summary
Este proyecto desarrolla una aplicación web interactiva que sustituye presentaciones tradicionales como PowerPoint. Facilita la creación de aplicaciones funcionales mediante herramientas de inteligencia artificial, orientada a usuarios sin experiencia en programación. El producto incluye una presentación completa con múltiples slides funcionales, componentes, estilos, y un template de Prompt Maestro descargable.

# Project Module Description
La aplicación cuenta con los siguientes módulos:
- **Contenido Programático**: Estructura de la charla con secciones detalladas.
- **Arquitectura de la Aplicación**: Descripción del stack tecnológico y estructura de carpetas.
- **Diseño de Interfaz y Experiencia**: Principios de diseño, paleta de colores, y componentes UI.
- **Storyboard de Interacción**: Flujo de navegación y animaciones.
- **Plan de Implementación Técnica**: Fases de desarrollo y tecnologías utilizadas.
- **Modelo de Datos**: Estructura de datos para slides y plantillas de prompts.
- **Notas del Presentador**: Guía y frases clave para la presentación.
- **Criterios de Éxito**: Indicadores de éxito para la charla y la aplicación.
- **Actividad Práctica**: Instrucciones para crear una aplicación usando IA.
- **Prompt para Demo**: Plantilla para generar una aplicación funcional con GitHub Copilot.

# Technology Stack
- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS, Shadcn-UI, Framer Motion.
- **Gestión de Estado**: React Context API, LocalStorage.
- **Deployment**: Compatible con Vercel, Netlify, GitHub Pages, y Azure Static Web Apps.

# Usage
Para instalar y ejecutar el proyecto:
1. Clona el repositorio.
2. Navega a la carpeta del proyecto.
3. Ejecuta `pnpm install` para instalar las dependencias.
4. Ejecuta `pnpm run dev` para iniciar el servidor de desarrollo.
